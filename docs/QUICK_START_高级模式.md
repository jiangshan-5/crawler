# 🚀 高级模式快速开始

## 3步启用高级模式

### 第1步：启动工具
```bash
run_universal_crawler.bat
```

### 第2步：勾选高级模式
在GUI中勾选 "🛡️ 高级模式"

### 第3步：自动安装（如需要）⭐ NEW!
- 如果弹出安装提示，点击【是】
- 等待1-2分钟自动安装
- 重启程序即可使用

**就这么简单！** 🎉

---

## 💡 使用示例

### 示例1: 爬取Flaticon图标（需要高级模式）

**配置**:
```
URL: https://www.flaticon.com/free-icons/weather
模式: 列表页面
列表容器: article.icon
☑️ 高级模式（必须勾选）
数量: 20
延迟: 3秒
```

**字段**:
- 名称: `h3, span.title`
- 图片: `img@src`
- 链接: `a@href`

**结果**: 成功爬取20个图标

---

### 示例2: 爬取GitHub（标准模式即可）

**配置**:
```
URL: https://github.com/trending
模式: 列表页面
列表容器: article.Box-row
☐ 高级模式（不需要）
数量: 10
延迟: 2秒
```

**字段**:
- 项目名: `h2 a`
- 描述: `p.col-9`
- 星标: `span.d-inline-block`

**结果**: 快速爬取10个项目

## ❓ 常见问题

**Q: 高级模式提示不可用？**
A: 运行 `pip install undetected-chromedriver`

**Q: 首次启动很慢？**
A: 正在下载ChromeDriver，只需等待一次

**Q: 何时用高级模式？**
A: 标准模式失败时，或网站有反爬虫保护时

**Q: 高级模式更慢？**
A: 是的，但成功率更高（95-99% vs 60-70%）
