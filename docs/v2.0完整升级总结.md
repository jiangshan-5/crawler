## 🎯 核心功能

### 高级爬虫模块
**文件**: `src/advanced_crawler.py`

**功能**:
```python
class AdvancedCrawler:
    def __init__(self, headless=True)      # 初始化
    def start()                             # 启动浏览器
    def fetch_page(url, wait_time=3)       # 获取页面
    def close()                             # 关闭浏览器
```

**特性**:
- 使用 undetected-chromedriver
- 支持无头模式
- 自动等待 JavaScript 渲染
- 上下文管理器支持

### 通用爬虫集成
**文件**: `src/universal_crawler.py`

**新增参数**:
```python
UniversalCrawler(
    base_url="https://example.com",
    use_advanced_mode=True  # 启用高级模式
)
```

**特性**:
- 透明切换标准/高级模式
- 智能降级（依赖缺失时）
- 统一的 API 接口
- 自动资源管理

**新增方法**:

**特性**:


## 🔧 技术实现

### 反反爬虫技术
```
undetected-chromedriver
    ↓
移除自动化特征
    ↓
模拟真实浏览器
    ↓
绕过反爬虫检测
```

### 自动安装流程
```
用户勾选高级模式
    ↓
检测依赖是否安装
    ↓
弹出三选一对话框
    ↓
后台线程执行 pip install
    ↓
实时显示安装进度
    ↓
安装完成提示重启
```

### 架构设计
```
UniversalCrawlerGUI (GUI层)
    ↓
UniversalCrawler (控制层)
    ↓
    ├─ requests (标准模式)
    └─ AdvancedCrawler (高级模式)
           ↓
       undetected-chromedriver
```
