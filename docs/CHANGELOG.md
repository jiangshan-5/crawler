## v2.0.2 (2026-03-27)

### 🐛 Bug修复
- **修复依赖检测问题**: 彻底解决安装后重启仍提示未安装的问题
  - 改进 `is_advanced_mode_available()` 为动态检测（每次调用都重新检测）
  - 移除复杂的模块重载逻辑，简化代码
  - 确保能立即检测到新安装的依赖
  - 详见: [Bug修复_依赖检测问题.md](Bug修复_依赖检测问题.md)

### 🔧 改进
- **Python 3.12+ 兼容性**: 自动安装流程现在会先安装 `setuptools`
  - Python 3.12+ 移除了 `distutils` 模块
  - `undetected-chromedriver` 依赖 `distutils`
  - 现在会自动安装 `setuptools` 提供兼容层

### 📦 依赖更新
- 新增 `setuptools>=65.0.0` 到 requirements.txt（Python 3.12+ 必需）

### 📖 文档更新

## 版本规则

### 版本号格式
`主版本号.次版本号.修订号`

### 版本号说明
- **主版本号**: 重大架构变更或不兼容更新
- **次版本号**: 新功能添加，向后兼容
- **修订号**: Bug修复和小改进

### 示例
- `1.0.0` → `2.0.0`: 添加高级模式（重大功能）
- `2.0.0` → `2.1.0`: 添加代理支持（新功能）
- `2.0.0` → `2.0.1`: 修复模块重载bug（Bug修复）

## 升级指南

### 从 v1.0 升级到 v2.0
1. 更新代码: `git pull`
2. 安装依赖: `pip install -r requirements.txt`
3. 启动工具: `run_universal_crawler.bat`
4. 勾选高级模式，点击【是】自动安装
5. 重启程序

### 从 v2.0 升级到 v2.0.1
1. 更新代码: `git pull`
2. 重启程序即可
