# 高级模式使用指南

## 🎉 新功能：高级模式（反反爬虫）

爬虫工具已升级，新增高级模式，能够：
- ✅ 绕过 Cloudflare、DataDome、Imperva 等反爬虫系统
- ✅ 支持 JavaScript 动态渲染内容
- ✅ 模拟真实浏览器行为
- ✅ 自动处理页面加载和等待

---

## 📦 安装步骤

### 方式1: 自动安装（推荐）⭐ NEW!

1. 启动爬虫工具
2. 勾选 "🛡️ 高级模式"
3. 在弹出的对话框中点击【是】
4. 等待自动安装完成（约1-2分钟）
5. 重启程序即可使用

**优点**: 一键安装，无需手动操作

---

### 方式2: 手动安装

#### 1. 安装依赖
```bash
pip install undetected-chromedriver
```

#### 2. 验证安装
```bash
python -c "import undetected_chromedriver; print('安装成功！')"
```

#### 3. 重启爬虫工具
关闭并重新打开爬虫GUI

---

## 🚀 使用方法

### 启动工具
双击 `run_universal_crawler.bat` 启动爬虫GUI

### 启用高级模式（自动安装）⭐ NEW!
1. 在"爬取控制"区域，勾选：
   ```
   ☑️ 高级模式（绕过反爬虫，支持JS渲染）
   ```

2. 如果未安装依赖，会弹出对话框：
   ```
   高级模式需要安装 undetected-chromedriver
   
   是否现在自动安装？
   
   点击【是】- 自动安装（推荐）
   点击【否】- 手动安装
   点击【取消】- 取消启用
   ```

3. 点击【是】，程序会自动安装依赖

4. 安装完成后，重启程序即可使用

### 状态指示
- ✓ 可用（绿色）- 高级模式已就绪
- ✗ 不可用（红色）- 需要安装依赖（点击勾选框自动安装）

## ⚠️ 注意事项

### 1. 首次使用高级模式
- 首次启动会自动下载ChromeDriver（约10MB）
- 需要等待几秒钟
- 下载完成后会自动缓存

### 2. 性能影响
- 高级模式会启动Chrome浏览器
- 内存占用约200-300MB
- 速度比标准模式慢3-5倍

### 3. 何时使用高级模式
**必须使用**:
- 标准模式无法获取数据
- 网站有反爬虫保护
- 内容通过JavaScript加载

**不建议使用**:
- 简单的静态网站
- 需要大批量快速爬取
- 资源受限的环境

### 4. 调试建议
如果高级模式也失败：
1. 增加请求延迟（5-10秒）
2. 减少爬取数量
3. 检查网络连接
4. 查看日志文件

## 🔧 故障排除

### 问题1: 提示"高级模式不可用"
**解决方案**:
- **方式1（推荐）**: 点击对话框中的【是】，自动安装
- **方式2**: 手动运行 `pip install undetected-chromedriver`

### 问题2: 启动高级模式时卡住
**原因**: 正在下载ChromeDriver
**解决方案**: 耐心等待，首次使用需要下载

### 问题3: 高级模式报错
**可能原因**:
- Chrome浏览器未安装
- ChromeDriver版本不匹配
- 防火墙阻止

**解决方案**:
1. 安装最新版Chrome浏览器
2. 重新安装依赖: `pip install --upgrade undetected-chromedriver`
3. 检查防火墙设置

### 问题4: 高级模式速度太慢
**优化方案**:
- 减少爬取数量
- 增加请求延迟（避免被封）
- 使用标准模式（如果网站支持）

## 🎓 技术原理

### 标准模式
```
requests → 获取HTML → BeautifulSoup解析
```
- 快速、轻量
- 只能获取静态HTML
- 容易被识别为爬虫

### 高级模式
```
undetected-chromedriver → 启动真实Chrome → 
执行JavaScript → 等待渲染 → 获取完整HTML → BeautifulSoup解析
```
- 模拟真实浏览器
- 移除自动化特征
- 绕过反爬虫检测
