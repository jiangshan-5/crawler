## 🎯 测试用例 1: 基础功能测试（小批量）

### 用例编号
TC-FLATICON-001

### 测试目的
验证爬虫基本功能是否正常，能否成功爬取少量图标

### 前置条件
- ✅ 已安装 Python 3.7+
- ✅ 已安装依赖库（requirements.txt）
- ✅ 网络连接正常
- ✅ 爬虫工具已启动

### 测试配置

#### 1. URL配置
```
目标 URL: https://www.flaticon.com/free-icons/weather
爬取模式: ⚫ 列表页面  ⚪ 单个页面
列表容器选择器: .icon--item
```

#### 2. 字段选择器配置
| 字段名 | CSS选择器 | 说明 |
|--------|-----------|------|
| 图标图片 | `img.icon--item__img` | 图标图片元素 |
| 图标标题 | `.icon--item__title` | 图标名称 |
| 图标链接 | `a.icon--item__link` | 图标详情页链接 |

#### 3. 爬取控制
```
爬取数量: 10 条
请求延迟: 2.0 秒
保存格式: ⚫ JSON  ⚪ CSV
保存位置: data/crawled_data
```

### 测试步骤

**Step 1: 启动工具**
1. 双击 `run_universal_crawler.bat`
2. 等待GUI界面打开

**Step 2: 填写URL配置**
1. 在"目标URL"输入框填入：
   ```
   https://www.flaticon.com/free-icons/weather
   ```
2. 选择"列表页面"模式
3. 在"列表容器选择器"填入：
   ```
   .icon--item
   ```

**Step 3: 配置字段选择器**
1. 点击"添加"按钮，添加第一个字段：
   - 字段名: `图标图片`
   - CSS选择器: `img.icon--item__img`
   - 说明: `图标图片URL`

2. 点击"添加"按钮，添加第二个字段：
   - 字段名: `图标标题`
   - CSS选择器: `.icon--item__title`
   - 说明: `图标名称`

3. 点击"添加"按钮，添加第三个字段：
   - 字段名: `图标链接`
   - CSS选择器: `a.icon--item__link`
   - 说明: `图标详情页`

**Step 4: 设置爬取控制**
1. 爬取数量设置为: `10`
2. 请求延迟设置为: `2.0` 秒
3. 保存格式选择: `JSON`
4. 保存位置填入: `data/crawled_data`

**Step 5: 开始爬取**
1. 点击"开始爬取"按钮
2. 观察进度条和日志输出
3. 等待爬取完成提示

**Step 6: 验证结果**
1. 打开 `crawler/data/crawled_data/` 目录
2. 检查是否生成 JSON 文件
3. 打开 JSON 文件查看数据结构

### 预期结果

✅ **成功标准：**
1. 工具正常启动，界面显示正常
2. 成功访问目标URL，无网络错误
3. 成功解析出10条图标数据
4. 生成的JSON文件包含以下字段：
   - 图标图片 URL
   - 图标标题
   - 图标链接
5. 日志显示"爬取完成"
6. 无异常错误信息

✅ **输出文件示例：**
```json

  {
    "图标图片": "https://cdn-icons-png.flaticon.com/512/xxx/xxx.png",
    "图标标题": "sunny weather",
    "图标链接": "https://www.flaticon.com/free-icon/sunny_xxx"
  },
  {
    "图标图片": "https://cdn-icons-png.flaticon.com/512/xxx/xxx.png",
    "图标标题": "rainy weather",
    "图标链接": "https://www.flaticon.com/free-icon/rainy_xxx"
  }
]
```
