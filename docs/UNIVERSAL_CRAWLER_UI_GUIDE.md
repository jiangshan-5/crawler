## 功能位置详解

### 1. 开始和停止按钮 ✅

**位置**：左侧配置区底部

```
┌─ 操作按钮 ─────────────────┐
│                             │
│  [🚀 开始爬取]  [⏹ 停止]   │
│                             │
└─────────────────────────────┘
```

**功能说明**：
- **🚀 开始爬取**：点击后开始爬取数据
  - 爬取开始后，此按钮会变为禁用状态
  - 停止按钮会变为启用状态
  
- **⏹ 停止**：点击后停止当前爬取
  - 初始状态是禁用的（灰色）
  - 只有在爬取过程中才能点击
  - 点击后会中止爬取任务

**使用流程**：
1. 配置好所有参数
2. 点击"🚀 开始爬取"
3. 观察日志和进度
4. 如需中止，点击"⏹ 停止"
5. 爬取完成后，按钮状态自动恢复

### 2. 保存位置选择 ✅

**位置**：左侧配置区 - 爬取控制部分

```
┌─ 爬取控制 ─────────────────────────┐
│                                     │
│  爬取数量: [10] 条                  │
│  请求延迟: [1.0] 秒                 │
│  ○ JSON  ○ CSV                     │
│                                     │
│  保存位置:                          │
│  ┌─────────────────────┬─────────┐ │
│  │ data/crawled_data   │ 浏览... │ │
│  └─────────────────────┴─────────┘ │
│                                     │
└─────────────────────────────────────┘
```

**功能说明**：
- **输入框**：显示当前保存位置
  - 默认：`data/crawled_data`
  - 可以直接输入路径
  - 支持相对路径和绝对路径

- **浏览...按钮**：打开文件夹选择对话框
  - 点击后弹出文件夹选择窗口
  - 选择任意文件夹作为保存位置
  - 选择后自动填入输入框

**使用方法**：

**方法 1：直接输入**
```
1. 点击"保存位置"输入框
2. 输入路径，例如：
   - D:\MyData\crawler_results
   - C:\Users\YourName\Documents\data
   - data/my_crawled_data
3. 按回车确认
```

**方法 2：浏览选择（推荐）**
```
1. 点击"浏览..."按钮
2. 在弹出的对话框中选择文件夹
3. 点击"选择文件夹"
4. 路径自动填入输入框
```

## 常见问题

### Q1: 我看不到开始和停止按钮？
**A:** 按钮在左侧配置区的最底部，在"爬取控制"部分下方。如果窗口太小，可能需要向下滚动或调整窗口大小。

### Q2: 停止按钮是灰色的，点不了？
**A:** 这是正常的！停止按钮只有在爬取进行中才能点击。点击"开始爬取"后，停止按钮会变为可用状态。

### Q3: 如何更改保存位置？
**A:** 有两种方法：
1. 直接在"保存位置"输入框中输入路径
2. 点击"浏览..."按钮，在对话框中选择文件夹

### Q4: 保存位置可以是任意文件夹吗？
**A:** 是的！可以选择任意文件夹，包括：
- 本地磁盘（C:\、D:\等）
- 网络驱动器
- 移动硬盘
- U 盘

### Q5: 如果保存位置不存在会怎样？
**A:** 程序会自动创建该目录。如果无法创建（如权限不足），会显示错误提示。

### Q6: 爬取过程中可以修改配置吗？
**A:** 不可以。爬取开始后，所有配置项都会被锁定。如需修改，请先点击"停止"按钮。

## 提示和技巧

### 提示 1：保存位置建议
- 使用有意义的文件夹名称
- 按日期或项目分类
- 避免使用中文路径（某些情况下可能有问题）

### 提示 2：测试配置
- 先设置较小的爬取数量（如 5 条）
- 测试选择器是否正确
- 确认数据格式符合预期
- 再增加爬取数量

### 提示 3：使用停止按钮
- 如果发现配置错误，立即点击停止
- 如果网站响应很慢，可以停止后调整延迟
- 如果数据不符合预期，停止后修改选择器

### 提示 4：保存位置管理
- 为不同项目创建不同文件夹
- 定期清理旧数据
- 备份重要数据
