## 🎯 核心功能说明

### 1. AdvancedCrawler 类
```python
class AdvancedCrawler:
    def __init__(self, headless=True)  # 初始化
    def start()                         # 启动浏览器
    def fetch_page(url, wait_time=3)   # 获取页面
    def close()                         # 关闭浏览器
```

**特性**:
- 使用 undetected-chromedriver 绕过反爬虫
- 支持无头模式（headless）
- 自动等待 JavaScript 渲染
- 上下文管理器支持（with语句）

## 💡 关键设计决策

### 1. 延迟导入
```python
try:
    import undetected_chromedriver as uc
    ADVANCED_MODE_AVAILABLE = True
except ImportError:
    ADVANCED_MODE_AVAILABLE = False
```
**原因**: 避免强制依赖，用户可选择是否安装

### 2. 智能降级
```python
if self.use_advanced_mode:
    if not is_advanced_mode_available():
        logger.warning("高级模式不可用，将使用标准模式")
        self.use_advanced_mode = False
```
**原因**: 提高容错性，依赖缺失时自动回退

### 3. 资源管理
```python
def close(self):
    if self.advanced_crawler:
        self.advanced_crawler.close()
```
**原因**: 防止浏览器进程泄漏

### 4. 上下文管理器
```python
with AdvancedCrawler() as crawler:
    soup = crawler.fetch_page(url)
```
**原因**: 自动资源清理，更安全
