## 如何使用

### 方法 1：双击启动（最简单）

```

```

### 方法 2：命令行启动

```bash
cd crawler
python src/gui_crawler.py
```

### 方法 3：先测试再启动

```bash
cd crawler
python test_gui.py    # 测试所有功能

```

## 使用步骤

### 第一步：启动 GUI
双击 `run_gui.bat` 或运行 `python src/gui_crawler.py`

### 第二步：配置参数
1. 勾选图标来源（推荐：Feather Icons + Heroicons）
2. 输入关键词（默认已填写：list、add、chart、settings）
3. 设置下载数量（默认：3）
4. 选择输出目录（默认即可）

### 第三步：开始爬取
点击 "🚀 开始爬取" 按钮，等待下载完成

### 第四步：应用到小程序
1. 点击 "📱 应用到小程序" 按钮
2. 选择来源：Feather
3. 输入图标名称：list add chart settings
4. 点击 "应用"

### 第五步：查看效果
```bash
cd ../accounting-miniapp
npm run dev:mp-weixin
```
在微信开发者工具中重新导入项目，查看新图标！

## 常见问题

### Q1: GUI 无法启动？
**A:** 运行 `python test_gui.py` 检查问题，通常是缺少依赖：
```bash
pip install -r requirements.txt
```

### Q2: 下载失败？
**A:** 
- 检查网络连接
- 只勾选 Feather Icons 和 Heroicons
- 关闭 VPN

### Q3: 应用失败？
**A:**
- 确保已经爬取了图标
- 检查小程序目录路径
- 确保图标名称正确

### Q4: 图标显示不正确？
**A:**
- 重新编译小程序
- 清除微信开发者工具缓存
- 重新导入项目
