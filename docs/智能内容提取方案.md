## 🎯 方案 1: Jina Reader API（推荐）⭐⭐⭐⭐⭐

### 简介
Jina Reader 是一个免费的 API，可以将任何网页自动转换为干净的 Markdown 格式。

### 特点
- ✅ **超级简单**: 只需在 URL 前加 `r.jina.ai/`
- ✅ **自动提取**: 自动识别主要内容，去除广告和导航
- ✅ **免费使用**: 每分钟 20 次请求（免费）
- ✅ **支持多种格式**: Markdown、JSON、HTML
- ✅ **支持 PDF**: 也可以提取 PDF 内容

### 使用方法

#### 方法 1: 浏览器直接访问
```
原始URL: https://www.example.com/article
转换URL: https://r.jina.ai/https://www.example.com/article
```

#### 方法 2: Python API
```python
import requests

url = "https://www.example.com/article"
response = requests.get(f"https://r.jina.ai/{url}")
markdown_content = response.text

print(markdown_content)
```

#### 方法 3: 带 API Key（更高限额）
```python
import requests

url = "https://www.example.com/article"
headers = {
    "Authorization": "Bearer YOUR_API_KEY"
}
response = requests.get(f"https://r.jina.ai/{url}", headers=headers)
markdown_content = response.text
```

### 优势
- 🚀 **零配置**: 不需要写选择器
- 🎯 **高准确度**: AI 自动识别主要内容
- 🔄 **自动适应**: 网站改版也不影响
- 📱 **支持动态内容**: 可以处理 JavaScript 渲染的页面

### 限制
- 免费版每分钟 20 次请求
- 需要网络连接（调用 API）

## 🎯 方案 2: Trafilatura（推荐）⭐⭐⭐⭐⭐

### 简介
Trafilatura 是一个 Python 库，专门用于从网页中提取主要文本内容。

### 特点
- ✅ **完全免费**: 开源，无限制
- ✅ **离线工作**: 不需要网络 API
- ✅ **高准确度**: 基于多种算法组合
- ✅ **多种格式**: 支持 Markdown、JSON、XML、TXT

### 安装
```bash
pip install trafilatura
```

### 使用方法

#### 基础用法
```python
import trafilatura

# 下载网页
downloaded = trafilatura.fetch_url('https://www.example.com/article')

# 提取主要内容
result = trafilatura.extract(downloaded)
print(result)
```

#### 提取为 Markdown
```python
import trafilatura

downloaded = trafilatura.fetch_url('https://www.example.com/article')

# 提取为 Markdown 格式
result = trafilatura.extract(
    downloaded,
    output_format='markdown',
    include_comments=False,  # 不包含评论
    include_tables=True      # 包含表格
)
print(result)
```

#### 提取为 JSON
```python
import trafilatura

downloaded = trafilatura.fetch_url('https://www.example.com/article')

# 提取为 JSON 格式（包含元数据）
result = trafilatura.extract(
    downloaded,
    output_format='json',
    with_metadata=True  # 包含标题、作者、日期等
)
print(result)
```

#### 批量提取
```python
import trafilatura

urls = [
    'https://www.example.com/article1',
    'https://www.example.com/article2',
    'https://www.example.com/article3'
]

for url in urls:
    downloaded = trafilatura.fetch_url(url)
    result = trafilatura.extract(downloaded, output_format='markdown')
    print(f"URL: {url}")
    print(result)
    print("-" * 50)
```

### 优势
- 🚀 **完全免费**: 无任何限制
- 🎯 **高准确度**: 专门优化过的提取算法
- 🔄 **离线工作**: 不依赖外部 API
- 📊 **提取元数据**: 自动提取标题、作者、日期

## 🎯 方案 3: ReadabiliPy

### 简介
基于 Mozilla Readability.js 的 Python 实现，专注于提取可读内容。

### 安装
```bash
pip install readabilipy
```

### 使用方法
```python
from readabilipy import simple_json_from_html_string
import requests

# 获取网页
response = requests.get('https://www.example.com/article')
html = response.text

# 提取内容
article = simple_json_from_html_string(html, use_readability=True)

print(article['title'])
print(article['plain_text'])
```

## 🎯 方案 4: AI 智能提取（未来方向）

### 使用 LLM 提取结构化数据

```python
import openai

def extract_with_ai(html_content, fields):
    """使用 AI 提取结构化数据"""
    prompt = f"""
    从以下 HTML 中提取这些字段：{fields}
    
    HTML:
    {html_content[:5000]}  # 限制长度
    
    以 JSON 格式返回结果。
    """
    
    response = openai.ChatCompletion.create(
        model="gpt-4",
        messages=[{"role": "user", "content": prompt}]
    )
    
    return response.choices[0].message.content
```

### 优势
- 🎯 **极高准确度**: AI 理解语义
- 🔄 **自动适应**: 完全不需要选择器
- 📊 **结构化输出**: 直接返回 JSON

### 限制
- 💰 **需要付费**: API 调用费用
- 🐌 **速度较慢**: 每次请求需要几秒
- 📏 **内容限制**: 有 token 限制

## 🚀 集成到现有爬虫

### 添加智能提取模式

```python
class SmartExtractor:
    """智能内容提取器"""
    
    def __init__(self, method='trafilatura'):
        """
        初始化
        
        Args:
            method: 提取方法 ('trafilatura', 'jina', 'readability')
        """
        self.method = method
    
    def extract(self, url):
        """智能提取内容"""
        if self.method == 'trafilatura':
            return self._extract_with_trafilatura(url)
        elif self.method == 'jina':
            return self._extract_with_jina(url)
        elif self.method == 'readability':
            return self._extract_with_readability(url)
    
    def _extract_with_trafilatura(self, url):
        """使用 Trafilatura 提取"""
        import trafilatura
        downloaded = trafilatura.fetch_url(url)
        return trafilatura.extract(
            downloaded,
            output_format='json',
            with_metadata=True
        )
    
    def _extract_with_jina(self, url):
        """使用 Jina Reader 提取"""
        import requests
        response = requests.get(f"https://r.jina.ai/{url}")
        return response.text
    
    def _extract_with_readability(self, url):
        """使用 ReadabiliPy 提取"""
        import requests
        from readabilipy import simple_json_from_html_string
        
        response = requests.get(url)
        return simple_json_from_html_string(
            response.text,
            use_readability=True
        )
```

```python
import hashlib
import os

def get_cached_or_extract(url):
    """使用缓存避免重复提取"""
    cache_key = hashlib.md5(url.encode()).hexdigest()
    cache_file = f"cache/{cache_key}.txt"
    
    # 检查缓存
    if os.path.exists(cache_file):
        with open(cache_file, 'r') as f:
            return f.read()
    
    # 提取内容
    content = smart_extract(url)
    
    # 保存缓存
    with open(cache_file, 'w') as f:
        f.write(content)
    
    return content
```

def validate_extraction(content):
    """验证提取的内容是否有效"""
    if not content:
        return False
    
    # 检查长度
    if len(content) < 50:
        return False
    
    # 检查是否包含常见的无效内容
    invalid_keywords = ['404', 'not found', 'access denied']
    if any(kw in content.lower() for kw in invalid_keywords):
        return False
    
    return True
