```python
def on_advanced_mode_toggle(self):
    """高级模式切换事件"""
    if not is_advanced_mode_available():
        # 弹出三选一对话框
        result = messagebox.askyesnocancel(
            "高级模式不可用",
            "是否现在自动安装？\n\n"
            "点击【是】- 自动安装（推荐）\n"
            "点击【否】- 手动安装\n"
            "点击【取消】- 取消启用"
        )
        
        if result is True:
            self.auto_install_advanced_mode()
```

#### `auto_install_advanced_mode()` - 新增
```python
def auto_install_advanced_mode(self):
    """自动安装高级模式依赖"""
    self.log("开始自动安装高级模式依赖...", "INFO")
    
    # 在后台线程中安装
    thread = threading.Thread(target=self._install_dependency)
    thread.daemon = True
    thread.start()
```

#### `_install_dependency()` - 新增
```python
def _install_dependency(self):
    """在后台线程中安装依赖"""
    import subprocess
    
    # 执行 pip install
    process = subprocess.Popen(
        [sys.executable, '-m', 'pip', 'install', 'undetected-chromedriver'],
        stdout=subprocess.PIPE,
        stderr=subprocess.PIPE,
        text=True
    )
    
    # 实时输出日志
    for line in process.stdout:
        self.log(f"  {line.strip()}", "INFO")
    
    # 检查结果
    if return_code == 0:
        self.log("✓ 安装成功！", "SUCCESS")
        messagebox.showinfo("安装成功", "请重启程序以使用高级模式")
    else:
        self.log("✗ 安装失败", "ERROR")
        messagebox.showerror("安装失败", "请手动安装")
```

## 📊 技术细节

### 使用的技术
- `subprocess.Popen` - 执行pip命令
- `threading.Thread` - 后台线程
- `messagebox.askyesnocancel` - 三选一对话框
- 实时stdout读取 - 显示安装进度

### 安全考虑
- 使用 `sys.executable` 确保正确的Python环境
- 使用 `-m pip` 而不是直接调用pip
- 捕获所有异常，防止程序崩溃
- daemon线程，程序关闭时自动结束

### 性能优化
- 后台线程不阻塞UI
- 实时输出避免假死感
- 安装完成后自动恢复按钮状态
