### 改进点 2：可选择保存位置
**状态**：✅ 已实现

**位置**：左侧配置区 - 爬取控制部分

**界面说明**：
```
保存位置:
┌─────────────────────────┬─────────┐
│ data/crawled_data       │ 浏览... │
└─────────────────────────┴─────────┘
```

**功能验证**：
1. 启动工具：`双击 run_universal_crawler.bat`
2. 查看"爬取控制"部分
3. 找到"保存位置"标签
4. 可以看到：
   - 输入框（显示当前路径）
   - "浏览..."按钮

**使用方法**：

**方法 1：直接输入路径**
```
1. 点击输入框
2. 清空现有路径
3. 输入新路径，例如：
   D:\MyData\crawler_results
   C:\Users\YourName\Documents\data
4. 按回车确认
```

**方法 2：浏览选择（推荐）**
```
1. 点击"浏览..."按钮
2. 在弹出的对话框中浏览文件夹
3. 选择目标文件夹
4. 点击"选择文件夹"
5. 路径自动填入输入框
```

**支持的路径类型**：
- ✅ 相对路径：`data/my_data`
- ✅ 绝对路径：`D:\MyData\crawler`
- ✅ 用户目录：`C:\Users\YourName\Documents`
- ✅ 网络驱动器：`\\server\share\folder`
- ✅ 移动设备：`E:\crawler_data`
