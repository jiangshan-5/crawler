# 🎉 新功能：一键自动安装高级模式

## 功能说明

现在用户无需手动安装依赖，只需在GUI中勾选"高级模式"，程序会自动提示并安装所需依赖！

---

## 🚀 使用流程

### 1. 启动工具
```bash
run_universal_crawler.bat
```

### 2. 勾选高级模式
在"爬取控制"区域勾选：
```
☑️ 高级模式（绕过反爬虫，支持JS渲染）
```

### 3. 自动安装提示
如果依赖未安装，会弹出对话框：

```
┌─────────────────────────────────────┐
│     高级模式不可用                   │
├─────────────────────────────────────┤
│ 高级模式需要安装                     │
│ undetected-chromedriver             │
│                                     │
│ 是否现在自动安装？                   │
│                                     │
│ 点击【是】- 自动安装（推荐）         │
│ 点击【否】- 手动安装                 │
│ 点击【取消】- 取消启用               │
└─────────────────────────────────────┘
```

### 4. 选择安装方式

#### 选项1: 自动安装（推荐）⭐
- 点击【是】
- 程序自动执行 `pip install undetected-chromedriver`
- 实时显示安装进度
- 安装完成后提示重启

#### 选项2: 手动安装
- 点击【否】
- 显示手动安装命令
- 用户自行在命令行安装

#### 选项3: 取消
- 点击【取消】
- 取消勾选高级模式
- 继续使用标准模式

### 5. 安装过程
点击【是】后，日志区域会显示：

```

==================================================
正在安装 undetected-chromedriver...
这可能需要几分钟，请耐心等待

==================================================
请重启程序以使用高级模式
```

### 6. 重启程序
安装成功后：
- 关闭当前程序
- 重新运行 `run_universal_crawler.bat`
- 勾选"高级模式"即可使用

## 💡 技术实现

### 核心代码
```python
def on_advanced_mode_toggle(self):
    """高级模式切换事件"""
    if self.advanced_mode_var.get():
        from advanced_crawler import is_advanced_mode_available
        if not is_advanced_mode_available():
            # 询问用户是否自动安装
            result = messagebox.askyesnocancel(
                "高级模式不可用",
                "是否现在自动安装？\n\n"
                "点击【是】- 自动安装（推荐）\n"
                "点击【否】- 手动安装\n"
                "点击【取消】- 取消启用"
            )
            
            if result is True:
                self.auto_install_advanced_mode()

def auto_install_advanced_mode(self):
    """自动安装高级模式依赖"""
    # 在后台线程中安装
    thread = threading.Thread(target=self._install_dependency)
    thread.daemon = True
    thread.start()

def _install_dependency(self):
    """在后台线程中安装依赖"""
    import subprocess
    
    process = subprocess.Popen(
        [sys.executable, '-m', 'pip', 'install', 'undetected-chromedriver'],
        stdout=subprocess.PIPE,
        stderr=subprocess.PIPE,
        text=True
    )
    
    # 实时输出日志
    for line in process.stdout:
        self.log(f"  {line.strip()}", "INFO")
```

### 关键特性
1. **后台安装**: 使用独立线程，不阻塞UI
2. **实时反馈**: 安装过程实时显示在日志区域
3. **错误处理**: 安装失败时提供详细错误信息
4. **用户友好**: 三种选项满足不同需求

## ⚠️ 注意事项

### 1. 网络要求
- 需要连接互联网
- 如果网络慢，安装时间会较长
- 建议在网络良好时安装

### 2. 权限要求
- 可能需要管理员权限
- 如果安装失败，尝试以管理员身份运行

### 3. 安装时间
- 正常情况: 1-2分钟
- 网络慢时: 3-5分钟
- 首次下载ChromeDriver: 额外1-2分钟

### 4. 磁盘空间
- 需要约50MB空间
- 确保有足够的磁盘空间

## 🐛 故障排除

### 问题1: 安装失败
**症状**: 显示"安装失败"错误
**原因**: 
- 网络问题
- 权限不足
- pip版本过旧

**解决方案**:
1. 检查网络连接
2. 以管理员身份运行
3. 更新pip: `python -m pip install --upgrade pip`
4. 手动安装: `pip install undetected-chromedriver`

### 问题2: 安装卡住
**症状**: 长时间无响应
**原因**: 网络慢或下载大文件

**解决方案**:
- 耐心等待（可能需要5-10分钟）
- 检查网络连接
- 重启程序重试

### 问题3: 安装后仍不可用
**症状**: 安装成功但高级模式仍显示不可用

**解决方案**:
- **必须重启程序**（这是关键！）
- 检查是否安装到正确的Python环境
- 运行测试: `python test_advanced_mode.py`
