## 📊 完整工作流程

### 1️⃣ 启动 GUI
```bash
双击 run_gui.bat
```

### 2️⃣ 配置参数
- 勾选：Feather Icons + Heroicons
- 关键词：list、add、chart、settings
- 数量：3
- 输出：默认

### 3️⃣ 开始爬取
点击 "🚀 开始爬取"，等待完成

### 4️⃣ 应用到小程序
- 点击 "📱 应用到小程序"
- 选择来源：Feather
- 输入图标：list add chart settings
- 点击 "应用"

### 5️⃣ 查看效果
```bash
cd accounting-miniapp
npm run dev:mp-weixin
```
在微信开发者工具中查看新图标！
