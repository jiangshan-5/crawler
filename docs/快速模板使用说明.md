# 🎯 快速模板使用说明

## 📖 什么是快速模板？

快速模板是预先配置好的网站爬取方案，让你无需手动配置选择器，一键应用即可开始爬取。

---

## 🚀 使用步骤

### 1. 启动工具

双击 `run_universal_crawler.bat` 启动完整版爬虫

### 2. 选择模板

在"URL 配置"区域，找到"🎯 快速模板"下拉框

选择你要爬取的网站，例如：
- Flaticon 图标网站
- GitHub 趋势
- Hacker News
- Product Hunt

### 3. 点击应用

点击【📋 应用】按钮

程序会自动：
- ✅ 填充示例URL
- ✅ 配置列表选择器
- ✅ 添加所有字段选择器

### 4. 开始爬取

直接点击【🚀 开始爬取】按钮即可！

---

## 💡 使用示例

### 示例1: 爬取 Flaticon 图标

1. 选择模板: "Flaticon 图标网站"
2. 点击【应用】
3. （可选）修改URL为你想爬的页面
4. 点击【开始爬取】

**自动配置**:
- 列表选择器: `li.icon--item`
- 字段: 图标名称、图标链接、图标图片、作者

### 示例2: 爬取 GitHub 趋势

1. 选择模板: "GitHub 趋势"
2. 点击【应用】
3. 点击【开始爬取】

**自动配置**:
- 列表选择器: `article.Box-row`
- 字段: 项目名、项目链接、描述、语言、星标数

## ❓ 常见问题

### Q: 模板不适用怎么办？

A: 网站可能更新了结构。你可以：
1. 手动调整选择器
2. 或使用"不使用模板"，手动配置

### Q: 可以添加自己的模板吗？

A: 可以！编辑 `src/website_templates.py` 文件，按照格式添加新模板。

### Q: 模板会自动更新吗？

A: 目前需要手动更新。如果网站结构变化，请调整选择器。

### Q: 为什么有些字段没有提取到？

A: 可能是：
1. 选择器不准确
2. 网站使用JavaScript动态加载（需要启用高级模式）
3. 字段在某些项目中不存在

## 🎯 最佳实践

### 1. 先用模板，再微调

不要从零开始配置，先应用最接近的模板，然后根据需要调整。

### 2. 测试小数量

第一次爬取时，设置爬取数量为 5-10 条，测试配置是否正确。

### 3. 保存配置

如果你经常爬取某个网站，可以：
1. 应用模板
2. 调整配置
3. 记录下来，下次直接使用

### 4. 启用高级模式

如果网站有反爬虫保护或使用JavaScript加载内容，记得勾选"高级模式"。
