## 🚀 如何使用

### 最简单的方式（推荐）⭐

1. **启动工具**
   ```bash
   run_universal_crawler.bat
   ```

2. **选择模板**
   - 在"🎯 快速模板"下拉框选择网站
   - 点击【📋 应用】按钮

3. **开始爬取**
   - 点击【🚀 开始爬取】
   - 10秒完成！
