## 技术实现

### 样式配置
```python
def setup_styles(self):
    """配置自定义样式"""
    style = ttk.Style()
    style.theme_use('clam')
    
    # 主按钮样式
    style.configure("Primary.TButton",
                   background=self.PRIMARY_COLOR,
                   foreground="white",
                   padding=(20, 10))
    
    # 表格样式
    style.configure("Treeview.Heading",
                   background=self.PRIMARY_COLOR,
                   foreground="white")
```

### 颜色常量
```python
PRIMARY_COLOR = "#2196F3"      # 主色调
SECONDARY_COLOR = "#00BCD4"    # 次要色
```
